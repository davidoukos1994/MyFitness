FitPlanner V1.5

FITPLANNER V1 — ΟΔΗΓΙΕΣ GITHUB PAGES

1. Αποσυμπίεσε το ZIP.
2. Ανέβασε όλα τα αρχεία στη ρίζα ενός GitHub repository.
3. Settings > Pages > Deploy from a branch.
4. Επίλεξε main και /(root), έπειτα Save.
5. Άνοιξε το link από Safari στο iPhone.
6. Κοινοποίηση > Προσθήκη στην οθόνη Αφετηρίας.

Τα δεδομένα αποθηκεύονται τοπικά στη συσκευή κάθε χρήστη.
Χρησιμοποίησε Εξαγωγή Backup για ασφαλή μεταφορά ή φύλαξη.
